You can seen in every page we have an option for search, reset, download pdf, excel export and excel import.These features are optional according to the User Profile.
●   SEARCH - Search button is just trying to find the information you need.
●   RESET - Reset button is an easy way to reset the form to their default values.
●   EXCEL EXPORT- The excel export allows exporting Grid data to Excel document in .xlsx form.
●   EXCEL IMPORT - The excel import allows importing Grid data from csv file then insert your data in the database and show your data in the grid.
●   DOWNLOAD PDF - PDF stands for "portable document format". Essentially, the format is used when you need to save files that cannot be modified but still need to be easily shared and printed.

IVR Information

IVR means Interactive voice response is used to make a selection for voice process. It allows the user to obtain information. We have DTMF selection in IVR-
● Digits - 0 to 9 , * , #
● Features -
     1. Call Group - IVR go through any selected call group
     2. Conference - IVR go through any selected conference
     3. Contact Exten - IVR go through selected contact
     4. Custom Exten - IVR go through any particular number which you have given in the value
     5. Hangup - IVR calls will be disconnected
     6. Queue - IVR go through any selected queue
     7. Repeat - IVR will repeat the calls
     8. SIP - IVR go through any selected SIP
     9. Voicemail - IVR go through the voicemail

Extension Information

Customer's may make any number of extensions but can't go beyond as per the provided package by the Administrator. Extensions are used for calling after registration with service with the help of IP phones, SIP softphone and mobile apps.
Customer can see all his Extensions at All Extension section. We may add new extensions after clicking at Add Button. There is an option of making single or multiple Extensions at once. But bundle extensions have some limitations like password would be the same as extension name etc. We create Extension by filling the mandatory fields with desirable features to its Extensions.

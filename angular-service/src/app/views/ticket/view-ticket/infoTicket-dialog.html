Ticket Information

If there is any issue, request or query, any user(admin,customer,internal user) will generate a ticket to his Existing Internal User( for now only assign account managers).
● The admin can change assignee to until there is no comment by the assigned assignee to but ticket status does not come in progress until assignee doesn't comment on it.
● Every time a unique Ticket is generated by a system like 2ET9511 (here 2 is ID and 9511 is number with ET as separator for this Ticket Number.)
● If we change the status of the ticket to CLOSE there nobody can comment on the existing ticket.
